# Calibri-Alt-G
A new generation of Calibri!

This is a slight modification of Calibri that uses stylistic alternatives to the default lowercase g. Like the original Calibri, it consists of three weights (Light, Regular, Bold), each with its italic variant. FontForge was used for its development.
